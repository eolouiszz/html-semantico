O que é um HTML semântico?

HTML semântico é o uso de tags que descrevem claramente o conteúdo da página, como <header>, <footer>, <article>, em vez de usar tags genéricas como <div> ou <span>. Isso melhora a acessibilidade e o SEO, tornando o código mais legível e estruturado.

Por que devemos usar HTML semântico?

Usar HTML semântico melhora a acessibilidade, facilita a manutenção do código, otimiza o SEO e torna a estrutura da página mais compreensível tanto para humanos quanto para máquinas (como buscadores e leitores de tela).

Como usar HTML semântico?

  • Cabeçalho da página: Use <header> em vez de <div>.
  • Artigo: Use <article> para conteúdo independente, como uma postagem de blog.
  • Rodapé: Use <footer> para o conteúdo final, como informações de contato ou direitos autorais.
  • Seção: Use <section> para dividir o conteúdo em partes relacionadas.
  • Navegação: Use <nav> para menus de navegação.
